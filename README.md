# travelguide

A React powered Travel Guide that makes travel simpler, enjoyable, and easy to access. The app fetches real-time data from an external API, offers smart travel suggestions, provides a virtual gallery for destinations, and includes an interactive map to explore nearby places.

## Features

- **API-Based Travel Data:** Fetches real-time information about destinations and travel tips.
- **Smart Travel Suggestions:** Helps users plan their trips with personalized recommendations.
- **Virtual Gallery:** View images of destinations to explore places visually.
- **Interactive Map:** Discover nearby places and travel spots.
- **Responsive UI:** Modern, clean, and mobile-friendly design.
- **Fast Performance:** Built with React + Vite for a smooth experience.

## Tech Stack

- **Frontend:** React, Vite,tailwindcss
- **Data:** External API (fetch)  
- **Maps:** leaflet
